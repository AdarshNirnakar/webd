types of positioning


As there are several different types of positioning.
I'll give you a quick summary.
There's relative positioning.
That means we can move the elephant relative to where it would otherwise be.
There's absolute positioning which takes the element out of the flow of the page and allows us to move it around relative to the top left of the page.
Fixed positioning does the same thing, but it keeps the element fixed even when we scroll down the page.
Sticky works like relative until scrolled past a specific threshold which is then treated as fixed and static, which is the default value that an element will have.
If you don't declare a position for an element and you may use to forcefully remove some other positioning.
And for all of those types, with the exception of static, we can use top, bottom, left and right properties to get it exactly where we want it.
